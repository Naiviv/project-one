ol

Back to Top

This element defines an ordered list, which can be organized numerical or alphabetical. The <li>tag defines it's list items.

  • reversed - Creates an numbered list with decending value (5, 4, 3...).
  • start - Identifies the start value of an ordered list.
  • type - Specifies the kind of mrker to use in the list (1, A, a, I, i).

<ol> This is an example of how this element is written in html </ol>

a

Back to Top

This tag defines a hyperlink.

  • href - Links a page's URL the the clickable link (most common use).
  • rel - Identifies the relationship between the two documents being linked.
  • target - Defines where to open the linked document.
  • type - Specifies the media type of the linked document.
  • download - Specifies that the target will be downloaded from this link.
  • hreflang - Identifies the language of the linked document.

<a attribute="value"> This is an example of how this element is written in html </a>

link

Back to Top

This tag defines a link between a document and an external source, links to external style sheets.

  • href - Specifies the location of the linked document.
  • rel - Identifies the relationship between the current document and the linked document.
  • type - Specifies the media type of the linked document.
  • hreflang - Identifies the language of the linked document.

<head> <link attribute="This is an example of how this element is written in html"> </head>
